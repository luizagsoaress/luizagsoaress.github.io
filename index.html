Sobre

Esse site é um projeto de estudo com HTML5, CSS, JavaScript, usando Chart.js para desenhar os gráficos, com integração de APIs de clima como a Open-Meteo e a OpenWeatherMap, cujas chaves são protegidas usando um backend em Node.js com Express. Para criação dos mapas interativos foi usado a biblioteca de código aberto Leaflet.
